# AI 開發指令（ai-dev-prompt.md）

你是一名資深全端工程師，
請依照以下規格協助開發一個教師培訓平台。

---

## 技術限制
- 前端：React + Tailwind CSS
- 後端：Supabase（Auth / Database / Storage）
- 登入：Google OAuth
- 架構：Headless CMS

---

## 必要功能
1. 教師登入與權限控管
2. 課程與章節顯示
3. 文章與影片內容顯示
4. 學習進度追蹤
5. 作業繳交（影片 / 文字）
6. 管理者後台
7. CMS（內容管理）

---

## CMS 規格
- 管理者可新增 / 編輯 / 下架：
  - 課程
  - 章節
  - 文章
  - 影片
- 內容可排序
- 內容有狀態（draft / published）

---

## UI 原則
- 簡潔
- 教育系統風格
- 強調進度與完成狀態
