# 後台管理員操作流程（admin-flow.md）

---

## 1. 登入流程
1. 管理者使用 Google 帳號登入
2. 系統檢查 role = admin
3. 導向管理者後台首頁

---

## 2. 課程管理流程
1. 進入「課程管理」
2. 新增課程（標題、描述、是否上架）
3. 調整課程排序
4. 下架或編輯既有課程

---

## 3. 章節管理流程
1. 點進某一課程
2. 新增章節
3. 調整章節順序
4. 設定章節是否顯示

---

## 4. 內容管理（CMS）
### 文章
1. 選擇章節
2. 新增文章
3. 編輯標題與內容
4. 設定狀態（草稿 / 發佈）

### 影片
1. 選擇章節
2. 新增影片
3. 填寫影片連結
4. 設定顯示狀態

---

## 5. 作業管理
1. 進入「作業管理」
2. 查看教師繳交作業
3. 播放影片或閱讀心得
4. 填寫回饋或評語

---

## 6. 進度總覽
1. 查看全體教師進度
2. 依課程 / 教師篩選
3. 作為培訓完成依據
